# pi-web 0.3.0

0.3.0 makes session navigation easier to share and resume, improves visual organization for long-running work, and expands rich message rendering.

## Highlights

### Session navigation and organization

- Tracks the active session in the URL so browser history, refreshes, and copied links preserve the selected session
- Adds default colors for session buckets so buckets are easier to scan without manual setup
- Dims inactive running session tabs to make the current workspace clearer
- Fixes drawer filtering behavior for pinned, bucketed, and active sessions

### Message rendering and review

- Renders Mermaid diagrams in markdown messages
- Improves flat edit diff rendering for clearer tool review

### README and release polish

- Refreshes README showcase content and screenshots for the current UI
- Carries forward the hardened release workflow from 0.2.x

## Intent

This release focuses on continuity: links should bring you back to the same work, session buckets should be easier to recognize, and richer markdown output should be useful without leaving pi-web.
